## Utilizando Ferramentas do GitHub para solucionar Algoritmos em Python
*   **1 - Concatenação de dados**
       Realiza a concatenação de duas strings de entrada.
*   **2 - Repetição de texto**
       Repete um texto fornecido por um número de vezes escolhido
*   **3 - Operações matemáticas**
        Realiza operações matemáticas simples (soma, subtração, multiplicação e divisão), recebendo dois números e a operação escolhida. Retorna o
        resultado da operação ou uma string esclarecendo o erro observado.
